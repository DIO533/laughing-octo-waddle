# 图表详细说明文档

## 概述

本文档详细解释再生制动仿真系统中各个图表的含义，特别是制动力分配比例图表和电池SOC变化图表。

## 1. 制动力分配比例图表

### 图表含义
制动力分配比例图表显示了在制动过程中，电机制动（再生制动）和液压制动（摩擦制动）各自承担的制动力比例。

### 关键概念

#### 电机制动（再生制动）
- **颜色**：绿色区域
- **含义**：电机作为发电机工作，将车辆动能转换为电能并存储到电池中
- **优点**：能量回收，提高效率，减少制动器磨损
- **限制**：受电机功率、电池SOC、车速等因素限制

#### 液压制动（摩擦制动）
- **颜色**：橙色区域  
- **含义**：传统的摩擦制动，将动能转换为热能并散失
- **特点**：制动力大，响应快，但能量完全损失
- **作用**：补充电机制动不足的部分

### 策略差异

#### 基准策略（固定6:4比例）
- 前轴制动力：60%（主要由电机承担）
- 后轴制动力：40%（液压制动）
- 电机制动占比相对固定，约58-60%

#### 优化策略（模糊逻辑控制）
- 根据车速、SOC、制动强度动态调整
- 低速轻制动：优先使用电机制动（可达90%+）
- 高速重制动：增加液压制动比例
- 高SOC时：减少电机制动，避免过充

### 图表解读要点

1. **绿色区域越大**：再生制动使用越多，能量回收效率越高
2. **橙色区域越大**：摩擦制动使用越多，能量损失越大
3. **平均电机占比**：整个工况下电机制动的平均比例
4. **制动事件分布**：不同时刻的制动力分配策略

## 2. 电池SOC变化图表

### 图表含义
SOC（State of Charge）图表显示电池荷电状态随时间的变化，反映能量回收的效果。

### 关键概念

#### SOC定义
- **SOC = 当前电量 / 电池容量 × 100%**
- 范围：0-100%
- 单位：百分比(%)

#### SOC变化的影响因素

1. **再生制动**：SOC上升（能量回收）
2. **电机驱动**：SOC下降（能量消耗）
3. **电池自放电**：SOC缓慢下降
4. **温度影响**：影响充放电效率

### 图表元素说明

#### SOC曲线（蓝色线）
- 显示整个工况过程中SOC的实时变化
- 上升段：再生制动回收能量
- 下降段：电机驱动消耗能量
- 平稳段：无明显能量交换

#### 关键水平线
- **红色虚线（90%）**：高SOC限制线
  - 超过此值时减少再生制动
  - 保护电池避免过充
- **红色虚线（20%）**：低SOC警告线
  - 低于此值时需要充电
  - 影响再生制动性能

#### 起始和结束标记
- **绿色圆点**：初始SOC
- **蓝色方块**：结束SOC
- **SOC变化量**：结束SOC - 初始SOC

### SOC变化分析

#### 正常情况
- **SOC增加**：再生制动回收的能量 > 消耗的能量
- **SOC减少**：消耗的能量 > 回收的能量
- **SOC稳定**：能量收支基本平衡

#### 影响因素
1. **工况特性**：制动频率、制动强度
2. **策略效果**：再生制动使用比例
3. **初始SOC**：影响再生制动的可用空间
4. **电池特性**：充电效率、容量
